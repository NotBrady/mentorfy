# Mentorfy

## Stack
- Next.js (App Router)
- React 19
- TypeScript (loose - use `any` liberally)
- Tailwind CSS
- framer-motion
- pnpm

## Conventions
- Use `'use client'` for components with interactivity
- Import paths use `@/` alias
- Inline styles preferred over CSS classes
- framer-motion ease arrays need `as const` for TypeScript
